Why I built Kulshan

Over six years working with AWS customers. The same pattern kept showing up.

Teams discuss solutions before understanding the baseline.

Kulshan began as the audit I kept performing manually. Cost Explorer, then cross-referencing, then documenting findings, then summarizing for the customer. Over time it became a CLI.

Run one command. Get one report. Start from facts.
