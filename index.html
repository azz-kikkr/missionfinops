Kulshan is the free diagnostic

Every engagement starts with the same question: What is actually happening in our AWS bill?

Kulshan is our open-source AWS assessment tool. It generates a local baseline report using read-only AWS APIs.

Kulshan is not the business. It is the free blood test before the real conversation.

Generates HTML and JSON reports locally using your existing AWS credentials.

No SaaS No telemetry No data uploads No write access
