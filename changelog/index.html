V0.1.0 shipped
2026-04-21 · Initial release

Ten audit packs, one CLI, one report.

The first version of Kulshan. Ten internal audit packs run through one command: cost, sweep, tag, age, dr, pulse, limit, drift, topo, security. A unified Kulshan report runs every pack against your AWS account and emits a Rich terminal dashboard with scores, severity breakdown, and JSON export.

Read-only AWS access from the start. The composed IAM policy ships with the source. Every action is Get, List, or Describe. The full policy and per-pack policies are at /policy.

The CLI ships with shell tab completion (bash, zsh, fish, powershell), interactive ? inline help, AWS profile tab completion that reads ~/.aws/config and ~/.aws/credentials, per-pack color themes, and Rich-formatted help output.

  • 10 audit packs in one CLI: cost, sweep, tag, age, dr, pulse, limit, drift, topo, security
  • Unified Kulshan report command across all packs
  • Rich terminal dashboard with score bars, severity breakdown, JSON export
  • Read-only IAM policy: 147 actions, 30 services, all Get/List/Describe
  • Shell tab completion: bash, zsh, fish, powershell
  • Interactive ? inline help and AWS profile completion
